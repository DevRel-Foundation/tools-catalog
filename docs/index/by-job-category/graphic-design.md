## Create Infographics

Infographics can convey complex information visually which aids with understanding. It may convey graphically relationships between objects, quantitative information summarized, etc.

This may include architecture diagrams, dataflow diagrams, E-R diagrams, and similar development-oriented tasks.

**Draw.io** | https://www.draw.io  
Draw.io is a free online diagramming tool that allows users to create flowcharts, network diagrams, UML diagrams, and other types of visual representations. It features a user-friendly interface with drag-and-drop functionality and supports collaboration through cloud storage or local filesystem integration. ([Source Data](../../../data/draw.io.json))
